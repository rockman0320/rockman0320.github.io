시작하기

앱 설치 후 첫 실행에서 기본 연결 정보를 입력하고, 테스트 연결을 통해 네트워크 접근 가능 여부를 먼저 확인하세요. 이 문단은 실제 매뉴얼 작성 전 구조 확인을 위한 placeholder 텍스트입니다.

연결 방법

새 연결 만들기에서 호스트, 포트, 사용자명을 입력한 뒤 비밀번호 또는 키 인증 방식을 선택합니다. 자주 쓰는 서버는 즐겨찾기로 등록해 다음 접속 시간을 줄일 수 있습니다.

설정

글꼴 크기, 터미널 색상, 자동 재연결 옵션 등을 환경에 맞게 조정합니다. 보안을 위해 민감한 인증 정보는 기기 보안 정책과 함께 관리하는 것을 권장합니다.

FAQ

연결이 지연되거나 실패한다면 네트워크 상태, 포트 오픈 여부, 인증키 권한 설정을 순서대로 점검하세요. 빈번한 문제와 해결 방법은 정식 문서 배포 시 업데이트될 예정입니다.
