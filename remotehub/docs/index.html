시작하기

앱 설치 후 초기 화면에서 프로필을 생성하고 기본 연결 대상을 등록합니다. 이 문서는 실제 콘텐츠 작성 전 구조 확인을 위한 placeholder 텍스트로 채워져 있습니다.

권한 설정

연결 요청 알림과 세션 동기화를 위해 알림, 네트워크, 백그라운드 실행 권한을 확인합니다. 제조사별 절전 정책에서 앱 예외 등록이 필요할 수 있습니다.

원격 요청

요청 목록에서 대상 기기를 선택하고 연결 승인 규칙을 적용합니다. 승인 이력은 상태 탭에서 확인할 수 있으며, 실패한 요청은 재시도 버튼으로 즉시 재전송합니다.

연결 문제 해결

연결이 끊기는 경우 네트워크 전환 시점, 토큰 만료 상태, 앱 버전 차이를 우선 점검하세요. 로그 코드 기준의 상세 진단 가이드는 이후 정식 문서에 반영됩니다.

FAQ

자주 묻는 질문과 운영 팁을 순차적으로 정리할 예정입니다. 앱 내 고객지원 채널을 통해 추가 질문을 접수할 수 있습니다.
